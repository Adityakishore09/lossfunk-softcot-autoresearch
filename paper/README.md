# Paper artifact

This folder contains the AI-authored first-draft paper for the selected Lossfunk autoresearch spike.

Files:

- `main.tex` - canonical LaTeX source.
- `references.bib` - bibliography with citation metadata checked against public arXiv/known source metadata where possible.
- `caisc_2026.sty` - CAISc style copied from `draft-format/`.
- `main.pdf` - fallback generated PDF, created because no local TeX compiler is installed.
- `build_pdf.py` - fallback PDF builder.
- `figures/` - generated PNG figures used by the fallback PDF and available for a future LaTeX compile.
- `rendered/` - rendered PDF page PNGs for visual QA.
- `COMPILE_NOTES.md` - build limitations and LaTeX rebuild commands.

The scientific claim is intentionally mixed/negative: the original related-transfer hypothesis was not supported.
